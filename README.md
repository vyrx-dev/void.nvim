## üìã Requirements

- Neovim >= 0.8.0
- termguicolors enabled

## üì¶ Installation

[lazy.nvim](https://github.com/folke/lazy.nvim)

```
return {
 { "vyrx-dev/void.nvim", 
    lazy = false,]
    priority = 1000 },
 {
  "LazyVim/LazyVim",
   opts = {
   colorscheme = "void",
  },
 },
}
```
